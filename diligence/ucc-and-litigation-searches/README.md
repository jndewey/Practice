# UCC and Litigation Searches

Skill and reference materials for reviewing UCC lien search results on behalf of a lender in commercial mortgage loan transactions.

## Contents

| File | Description |
|------|-------------|
| `UCC_Search_Review_Skill_Commercial_Mortgage_Lending.md` | Systematic framework for ordering, reviewing, categorizing, and dispositioning UCC search results, with a focus on commercial mortgage lending considerations. |

## Key Topics Covered

- Why UCC searches matter in commercial mortgage lending (prior liens, negative covenants, permitted liens)
- What searches to order (core UCC, federal/state tax liens, judgment liens, fixture filings, bankruptcy, ERISA, IP)
- Ensuring the correct debtor name (exact legal name, trade names, prior names, tax return name per *Spearing Tool*)
- How to review UCC-1 and UCC-3 filings (secured party, collateral description, filing date, amendments, terminations)
- Categorizing and dispositioning each filing:
  - Terminated/Lapsed
  - Permitted Lien
  - Informational/Precautionary Filing
  - Must Be Terminated at Closing
  - Requires Further Investigation
- Preparing the results chart (searches ordered, filings found, dispositions)
- Key issues specific to commercial mortgage loans (fixtures, rents/leases, deposit accounts, SPE borrowers, mezzanine lenders)
- Timing considerations (initial search, bring-down, post-closing)
- Common pitfalls (wrong name, lapsed filings, rogue amendments, missing fixture filings)
- Deliverables checklist for pre-closing confirmation
