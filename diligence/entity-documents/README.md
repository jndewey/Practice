# Entity Documents Due Diligence

Skill and reference materials for conducting borrower-party entity due diligence in commercial real estate loan transactions, oriented toward lender's counsel.

## Contents

| File | Description |
|------|-------------|
| `reviewing entity documents for senior lender.md` | Comprehensive skill covering SPE requirements, bankruptcy remoteness, organizational document checklists by entity type (LLCs, corporations, partnerships, joint ventures), CTA compliance, formation verification, good standing, foreign qualification, power and authority, and closing deliverables. |

## Key Topics Covered

- Pre-diligence review of loan terms (SPE, bankruptcy remoteness, securitization triggers)
- Single Purpose Entity (SPE) requirements and core characteristics
- Bankruptcy remote structure (independent director/manager requirements)
- Corporate Transparency Act (CTA) compliance for non-US equity owners
- Organizational document checklists by entity type
- Formation verification and good standing certificates (30-day staleness rule)
- Foreign qualification requirements
- Power and authority confirmation
- Closing deliverables (secretary's certificate, omnibus certification)
- Quick reference closing checklist with responsible parties and staleness periods
